3. Identifique a finalidade dos seguintes comandos:

a. git init<br/> : Cria um repositório local.
b. git config --global user.name "turing"<br/> : Define o seu nome de usuário no projeto.
c. git add EXERCICIO.txt<br/> : Adiciona o arquivo na área de staged
d. git add .<br/> : Adiciona as modificações.
e. git commit -m "Adicionado nova interface"<br/> : Consolida todos os arquivos e adiciona uma mensagem.
f. git commit<br/> : Apenas consolida os arquivos.
g. git reset --hard HEAD<br/> : Desfaz o commit e todas as alteraões.
h. cd Downloads<br/> : Vai para a pasta "Downloads".
i. pwd<br/> : Mostra o diretório atual.
j. cd ..<br/> : Retorna ao diretório de origem.
k. ls<br/> : Mostra todo o conteúdo disponível no diretório.
l. git pull<br/> : Sincroniza os arquivos do repositório local com o remoto.
m. git push<br/> : Envia atualizações para o servidor remoto.
n. git clone https://github.com/wiecodepython/Exercicios.git<br/> : Baixa um projeto do link desejado.
o. git diff<br/> : Mostra as diferenças entre o repositório local e o remoto.
p. git show<br/> : Exibe os detalhes do commit atual.
